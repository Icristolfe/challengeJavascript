# Javascript
## Como rodar os testes

No terminal, execute os comandos:

```bash
cd javascript
yarn
yarn test
```

Ou usando o NPM:

```bash
cd javascript
npm install
npm test
```

Em caso de erro instale o jest manualmente:
 Yarn:
```bash
yarn add --dev jest
```

Npm:
```bash
npm install --save-dev jest
```
